Allen-Bradley ControlLogix Fault Codes

ControlLogix (1756 series) is the workhorse of large-scale industrial automation. Major faults halt the controller and require intervention. Minor faults log warnings but keep running. Common ControlLogix fault types include:

  • Type 01 — Power-Up: Controller powered up without a project or with a corrupted program. Download the project from RSLogix 5000.
  • Type 04 — Watchdog: A task exceeded its watchdog timer. Usually means a program loop or excessive processing. Increase the watchdog or optimize the routine.
  • Type 06 — Instruction: Math overflow, divide by zero, or illegal instruction. Check the faulted routine in RSLogix.
  • Type 11 — Motion: Servo axis faults — position error, overcurrent, or drive fault. Check the Kinetix drive status and axis properties.

PowerFlex VFD Fault Codes

PowerFlex drives (40, 400, 525, 755) are Allen-Bradley's variable frequency drive line. Faults display on the drive HMI and propagate to the PLC via EtherNet/IP. Common PowerFlex faults:

  • F002 — Auxiliary Input: External fault input triggered. Check the wiring to the drive's digital inputs.
  • F004 — Undervoltage: DC bus voltage dropped below threshold. Check incoming power, connections, and capacitor bank.
  • F005 — Overvoltage: DC bus voltage exceeded limit during deceleration. Add a braking resistor or increase decel time.
  • F007 — Motor Overload: Electronic overload tripped. Verify motor FLA, check for mechanical binding.
  • F012 — HW Overcurrent: Instantaneous overcurrent detected. Check for output short circuits or ground faults.

GuardLogix Safety Fault Codes

GuardLogix safety controllers require SIL 2/SIL 3 compliant fault handling. Safety faults lock out the safety task and require a deliberate reset. Key fault types:

  • Safety Partner Missing: The safety partner (1756-L7SP) lost communication with the primary controller. Check the backplane connection.
  • Safety Task Fault: The safety routine faulted. Unlike standard tasks, safety faults cannot be auto-cleared. Review the safety routine logic.
  • Guard I/O Fault: A POINT Guard I/O module lost its safety connection. Verify the safety connection parameters and CIP Safety timeout.

Allen-Bradley Fault Code FAQ

What are the most common Allen-Bradley ControlLogix fault codes?

The most common ControlLogix fault codes include: Major Fault Type 01 (Power-up default), Type 04 (Watchdog fault), Type 06 (Instruction fault), and Type 11 (Motion fault). RepairXpert covers 74 Allen-Bradley fault codes with step-by-step repair procedures.

How do I clear a major fault on an Allen-Bradley PLC?

Connect via RSLogix 5000 / Studio 5000. Check Controller Properties > Major Faults to identify the fault type. Fix the underlying cause, then click 'Clear Major Fault' in the Controller menu and switch back to Run mode.

What does PowerFlex fault code F004 mean?

F004 is an Undervoltage fault — the DC bus voltage dropped below the minimum threshold. Check incoming power, verify wire terminations, look for voltage dips during motor starting, and inspect DC bus capacitors for bulging.

How do I troubleshoot EtherNet/IP communication faults?

Check physical connections and LEDs. Verify IP addresses and subnets. Check RPI settings. Use RSLinx to confirm the module path. Review 1756-EN2T/EN3TR status LEDs: green = OK, red = fault, flashing = no connection.
