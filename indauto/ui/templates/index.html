Quick Help — Before You Call Maintenance

  • ⚠ Check all E-stops — Walk the entire machine and twist/pull every E-stop button before calling maintenance. Most "machine won't run" calls are a bumped E-stop.
  • ⏻ Power cycle the PLC if comms are down — Turn the PLC off, wait 10 seconds, turn it back on. This clears most communication faults with remote I/O and drives.
  • 💡 Verify sensor LEDs before replacing — A sensor with its LED on but not triggering the PLC usually means a wiring or input card issue, not a bad sensor.
  • 🔧 Check air pressure first — Low plant air causes random pneumatic faults across the whole machine. Check the FRL gauge before troubleshooting individual cylinders.
  • 📜 Note the exact fault code — Write down the fault code from the HMI or VFD display before resetting. The code disappears after reset and maintenance needs it.
