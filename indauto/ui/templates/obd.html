      About OBD-II Scanner

      This tool reads OBD-II diagnostic data from any 1996+ vehicle. Currently running in demo mode with realistic test data. Connect an ELM327 adapter via USB or Bluetooth for live vehicle data.

      MCP Integration: This is the first automotive OBD-II MCP server. AI assistants can read codes, pull live data, and get expert diagnosis through the MCP protocol. Add repairxpert-obd to your MCP config to enable AI-powered vehicle diagnostics.

      Pair with VIN Decoder for full vehicle identification, or AI Diagnosis for industrial equipment faults.
